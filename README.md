# Global Equity Intelligence Dashboard

A lightweight web dashboard for institutional-style stock analysis across global equities.

## What it evaluates
- Fundamental quality: growth, margins, ROIC, leverage, and revenue-to-profit conversion.
- Valuation: forward P/E against sector median plus FCF yield context.
- Technicals: 200DMA trend, crossover signals, RSI, volatility-adjusted entry/exit planning.
- Macro + business context: industry growth and company product growth relative to industry.
- AI/ML-style blended recommendation: confidence score, buy/accumulate/watch/avoid regime.
- 1Y/2Y/3Y price projections adjusted by guidance confidence and risk discount.

## Run locally
Open `index.html` directly in browser, or run:

```bash
python3 -m http.server 4173
```
Then open `http://localhost:4173`.
